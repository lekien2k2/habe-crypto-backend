🔐 Mã hóa Phân cấp (HABE)

Hierarchical Attribute-Based Encryption — CP-ABE + AWS S3 Tiered Storage

🎯 Bài toán

Trong tổ chức phân cấp, cần kiểm soát truy cập dữ liệu dựa trên vai trò và thuộc tính:

  • Manager Dept A → xem tài liệu Dept A
  • Admin → xem tất cả
  • Employee Dept B → không xem được tài liệu mật Dept A

→ Chính sách truy cập được nhúng vào ciphertext, không phụ thuộc server.

🔑 CP-ABE (BSW07)

  • 🔒 Encrypt: Chỉ định policy → "MANAGER and DEPTA"
  • 👤 User key: Gắn attributes → ["MANAGER", "DEPTA"]
  • ✅ Decrypt: Thành công khi attributes ⊇ policy
  • ❌ Denied: Thất bại khi attributes ⊄ policy
Hybrid: ABE encrypt key → AES encrypt data → HMAC integrity

🛡️ Security Layers

🔐L1: CP-ABE — access control nhúng trong ciphertext
🔑L2: AES-256-CBC — mã hóa dữ liệu nhanh
🛡️L3: HMAC-SHA256 — integrity verification
☁️L4: S3 SSE — server-side encryption at rest
🚪L5: IAM — least privilege (PutObject + GetObject)
